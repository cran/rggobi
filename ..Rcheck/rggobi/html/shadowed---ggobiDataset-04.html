"shadowed<-.ggobiDataset" {rggobi}R Documentation

Set shadowed status

Description

Set the exclusion status of points.

Usage

"shadowed<-.ggobiDataset"(x, value)

Arguments

x ggobiDataset
value logical vector

Details

If a point is shadowed it is drawn in a dark gray colour.
