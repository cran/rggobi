ggobi_longitudinal {rggobi}R Documentation

Create longitudinal dataset.

Description

Instantiate new ggobi with a longitudinal data set.

Usage

ggobi_longitudinal(data, time = "Time", obsUnit = "ID")

Arguments

data data frame
time time variable
obsUnit id variable

Details

@arguments data frame @arguments time variable @arguments id variable @keyword dynamic

Author(s)

Hadley Wickham <h.wickham@gmail.com>

Examples

data(Oxboys, package="nlme")
ggobi_longitudinal(Oxboys, "Occasion", "Subject")
